# Launchpad (static demo)

Vite + React app with in-memory data (`src/api/launchpadMock.js`). There is no live API, Hub OAuth, or session backend in this build.

- `npm install --include=dev` — devDependencies are required to build
- `npm run dev` — local development
- `npm run build` — production assets in `dist/`

Copy `.env.example` to `.env` only if you need to override defaults.
